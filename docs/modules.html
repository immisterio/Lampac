Расширяемость

Пользовательские модули

Roslyn компилирует .cs-файлы при запуске. При "dynamic": true — горячая пересборка при изменении.

📁

Структура

Создайте подкаталог в mods/ с manifest.json и .cs-файлами. Сервер скомпилирует их при старте.

⚡

Горячая перезагрузка

При "dynamic": true в manifest.json изменения .cs-файлов применяются без перезапуска сервера.

🔌

Интерфейсы

Реализуйте IModuleConfigure для регистрации в DI и IModuleLoaded для вызова после старта приложения.

mods/MyModule/manifest.json
{
  "name":        "MyModule",
  "description": "Описание модуля",
  "version":     "1.0",
  "enable":      true,
  "dynamic":     true
}
