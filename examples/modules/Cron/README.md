# Cron (пример)

Модуль **`Lamson`** демонстрирует два типа фоновой работы при **`Loaded`**:

1. **`ThreadPool.QueueUserWorkItem`** — бесконечный цикл с **`await Task.Delay(1000)`** (условный воркер).
2. **`System.Timers.Timer`** с **`AutoReset`** раз в **1 минуту** — условный cron (внутри снова **`Task.Delay`**).

## Важно

- В **`Dispose`** обработчики **не** снимаются и таймер **не** останавливается — для учебного кода это сознательное упрощение; в реальном модуле нужно сохранять ссылку на **`Timer`**, отписываться от **`Elapsed`** и отменять долгие циклы через **`CancellationToken`**.
- Бесконечный **`while (true)`** без отмены при выгрузке приложения оставляет фоновую задачу до завершения процесса.

## manifest.json

Без **`dynamic`** — только **`ModInit.cs`** в **`tree`**.
