# PingPong (пример)

Минимальный модуль **без `ModInit.cs`**: один контроллер **`PingPongController`**, наследник **`BaseController`**.

## HTTP

| Маршрут | Ответ |
|---------|--------|
| **GET** `lamson/ping` | Текст **`pong`** (`Content`). |

## manifest.json

**`dynamic`: true**, в **`tree`** только **`Controller.cs`** — хост подхватывает класс при динамической загрузке.

## Назначение

Показать минимальную точку входа модуля без конфигурации и без жизненного цикла **`IModuleLoaded`**.
