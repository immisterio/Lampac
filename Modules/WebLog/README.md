# WebLog

Интерактивный просмотр **HTTP-трафика сервера** и ответов **Playwright** в реальном времени через WebSocket: отдельная страница UI и регистрация клиента по паролю root.

## Назначение

- Помогает отлаживать запросы к внешним API, прокси и онлайн-балансерам без постоянного чтения файловых логов.
- После успешной регистрации клиента модуль подписывается на глобальные события `HttpResponse` и `PlaywrightHttpResponse` и рассылает текстовые дампы всем подключённым «слушателям».

## Веб-интерфейс

| Маршрут | Доступ | Описание |
|---------|--------|----------|
| `GET /weblog` | По правилам `[Authorization]` модуля | Основная страница WebLog (`index.html` из каталога модуля). |
| `GET /weblog/auth` | Анонимно | Страница авторизации (`auth.html`). |

Класс контроллера помечен `[Authorization(redirectUri: "/weblog/auth")]` — точное поведение входа задаётся общими атрибутами Lampac.

## Работа через WebSocket (NWS)

1. Клиент подключается к общему WebSocket-серверу приложения.
2. Отправляется сообщение с методом **`registryweblog`** и первым аргументом — **`CoreInit.rootPasswd`** (пароль root Lampac).
3. При совпадении пароля `connectionId` заносится в список получателей; при отключении — удаляется.
4. Пока есть хотя бы один зарегистрированный клиент, включается трансляция:
   - канал **`http`**: форматированный вывод обычных HTTP-ответов (метод, URL, тело POST, заголовки, код ответа, тело ответа);
   - канал **`playwright`**: запросы/ответы из автоматизации браузера (Chromium).

Если клиентов нет, подписки на события снимаются (меньше накладных расходов).

## Конфигурация

Отдельной секции в `init.conf` у модуля **нет** — достаточно загрузить модуль (`manifest.json`: `enable`).

## Зависимости

- `Shared`: события `EventListener`, `Startup.Nws` для рассылки.
- Статика: `index.html`, `auth.html` в каталоге модуля.

## Файлы

| Файл | Роль |
|------|------|
| `LogEvents.cs` | Регистрация в NWS, подписки на HTTP/Playwright, broadcast. |
| `ApiController.cs` | Раздача HTML. |
| `ModInit.cs` | Старт/остановка `LogEvents`. |
