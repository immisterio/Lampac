Группы (JSON)

Ключи из current.conf, сгруппированные по секциям. Правки сохраняются в init.conf. Удалить ключ: используйте null или редактируйте весь файл.

Простой редактор

Поля формы по структуре current.conf + init.conf. Значения берутся из init, подсказки — из current. Массивы и вложенные объекты — JSON-блоки.
