# Kit

Минимальный **служебный модуль** без HTTP и без **`EventListener`**: единственная задача — сохранить путь к каталогу модуля на диске, который передаёт хост при загрузке.

## Поведение

| Этап | Действие |
|------|----------|
| **`Loaded(InitspaceModel conf)`** | **`KitMod.ModInit.folder_mod = conf.path`** |
| **`Dispose()`** | Пустой (состояние не сбрасывается) |

**`folder_mod`** — публичное статическое свойство с **приватным** сеттером; снаружи его можно только читать после **`Loaded`**.

## Пространство имён

**`KitMod`**, класс **`ModInit`**, интерфейс **`IModuleLoaded`**.

## Зачем это нужно

Другие модули или внешние сборки, ссылающиеся на **Kit**, могут резолвить файлы рядом с **`module/Kit/`** без хардкода абсолютного пути — достаточно прочитать **`KitMod.ModInit.folder_mod`**.

## Конфигурация

Секции в `init.conf` **нет** — только **`manifest.json`** (флаг **`enable`**).

## Зависимости

Нет. Важно: не обращайтесь к **`folder_mod` до** вызова **`Loaded`**, иначе значение будет **`null`**.
