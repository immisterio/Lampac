Подсказки по полям

enable — true (включить источник) / false (отключить)
corseu — true (прокси) / false (без прокси)
host — зеркало или API источника
scheme — http / https
pro / premium — подписка
hls — true (для современных ТВ) / false (mp4)
filetype — hls / hls4 / mp4
